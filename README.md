# Expense Tracker

A console-based Expense Tracker built using Python and MySQL.

## Features
- Add Income
- Add Expenses
- View Transactions
- Generate Monthly Report
- Calculate Balance

## Technologies Used
- Python
- MySQL
- VS Code
- Git & GitHub

## How to Run
1. Create the database using database.py
2. Create tables using create_tables.py
3. Run main.py
4. Choose options from the menu

## Project Structure
- add_income.py
- add_expenses.py
- view_transactions.py
- monthly_report.py
- create_tables.py
- database.py
- main.py
